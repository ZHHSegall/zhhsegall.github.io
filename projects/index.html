Current Projects

Genderless in Germany

This is a research project in Spring 2017 to build a visual novel that integrates a game engine with an intelligent tutoring system. The motivation behind the project is to create a fun and engaging game to teach German (educational games often fall short). The premise is that the player is a genderless alien who has crash landed in Germany and needs to fix their ship and get home, but is quite confused by the German language, particularly all the gendered pronouns. We will be developing the project in Unity.

My role: Project Manager
