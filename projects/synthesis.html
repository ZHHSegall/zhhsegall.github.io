--- layout: project title: Semi-Automated Program Synthesis ---

Program synthesis is the process of generating a program from specifications. Our approach is using a type theory and proof search.

The Motivation: The type theoretic model experiences an exponential blow up in the search space.

The Research: Our project is expanding the automated MYTH synthesizer into a semi-automated synthesizer that allows the user to guide the synthesis process, reducing the search space with each choice

The Tools: emacs, OCaml, GitHub

My role: Research Assistant
